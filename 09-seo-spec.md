## Meta Description Rules

Each meta description should be 140 to 160 characters when possible and include:

- company name
- service
- Florida or target city
- trust or quality phrase

## H1 Rules

Each page gets only one H1.

## Local SEO

Add service area text to footer:

> Proudly serving homeowners across Northeast and Central Florida, including Volusia County, Flagler County, St. Johns County, Putnam County, and surrounding areas.

Adjust counties and cities to match actual operating area.

## Schema Markup

Add LocalBusiness schema with:

- name
- url
- telephone
- email
- address if available
- service area
- sameAs social links
- logo
- image
- description

Use `HomeAndConstructionBusiness` or `LocalBusiness`.

## Google Business Profile

Website must match GBP name:

> Southern Cypress Homes

Use same phone and service areas.
