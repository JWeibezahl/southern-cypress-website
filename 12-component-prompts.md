# GitHub Copilot Component Prompts

## Header Prompt

Create a responsive header for Southern Cypress Homes. Include a dark green top contact bar with phone, email, and service area. Below it, include a cream navigation bar with the logo on the left, navigation links in the center, and a gold Get a Quote button on the right. On mobile, collapse navigation into a hamburger menu. Use accessible buttons and semantic HTML.

## Hero Prompt

Create a homepage hero section using a warm kitchen and outdoor living background image. Add a dark green gradient overlay on the left. Include eyebrow text "Quality. Integrity. Craftsmanship.", headline "Renovations That Elevate the Way You Live.", short body copy, and two CTA buttons: Get a Quote and View Our Work.

## Service Strip Prompt

Create a horizontal service strip with six cards: Kitchen Remodeling, Bathroom Remodeling, Outdoor Living, Decks & Pergolas, Exterior Painting, Whole Home Updates. Each card should include a simple line icon placeholder, title, and short description. Make it responsive and stack on mobile.

## About Section Prompt

Create an about preview section with a dark green background, serif headline, body text, and three project image cards. Use the heading "Crafted With Purpose. Built Around You." Include copy about quality residential renovations, clear communication, and lasting value.

## Featured Work Prompt

Create a portfolio grid with four cards for kitchen remodel, bathroom remodel, outdoor living, and custom deck. Each card should have an image, service title, short location placeholder, and hover effect.

## Trust Section Prompt

Create a trust/value section with four icons and short text: Quality Craftsmanship, Trusted Trade Partners, Clear Communication, On Time and On Budget. Include the trade partner language without implying licensed GC work.

## Footer Prompt

Create a dark green footer with the Southern Cypress Homes logo, tagline, quick links, services list, contact info, service areas, and a Get a Quote CTA box.

## Contact Form Prompt

Create a contact page with a project request form. Include fields for first name, last name, phone, email, project address, project type, project timeline, estimated budget, message, and photo upload placeholder. Include a spot to embed a HubSpot form later.
